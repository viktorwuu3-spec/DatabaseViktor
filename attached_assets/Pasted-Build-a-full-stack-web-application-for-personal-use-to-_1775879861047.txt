Build a full-stack web application for personal use to manage purchase data and purchase planning data.

TECH STACK:

* Backend: Node.js with Express.js
* Database: SQLite (use better-sqlite3 or sqlite3)
* Frontend: HTML, CSS (use Tailwind CSS), and vanilla JavaScript
* Export features: Use libraries like pdfkit (for PDF) and xlsx (for Excel)

DATABASE STRUCTURE
==================

Create two tables:

1. purchases
2. purchase_plans

Each table fields:

* id (INTEGER PRIMARY KEY AUTOINCREMENT)
* nomor (TEXT)
* tanggal (TEXT)
* keterangan (TEXT)
* jumlah (REAL)
* satuan (TEXT)
* harga_satuan (REAL)
* harga_total (REAL)
* catatan (TEXT)

==================================================
BACKEND REQUIREMENTS
====================

1. Setup Express server in server.js

2. Connect SQLite database in /config/db.js

3. Create REST API:

   * GET all data
   * POST new data
   * PUT update data
   * DELETE data

4. Separate logic:

   * Models в†’ database queries
   * Controllers в†’ logic
   * Routes в†’ endpoints

5. Auto calculate:
   harga_total = jumlah * harga_satuan
